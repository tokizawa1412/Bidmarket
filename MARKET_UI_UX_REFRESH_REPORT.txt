- ไม่แก้ฐานข้อมูล

4. Category chips สำหรับกรองรายการบน client side
5. Sort dropdown สำหรับเรียงล่าสุด/ราคาต่ำ/ราคาสูง/ผู้ขาย
6. Card สินค้าแบบใหม่ รูปใหญ่ขึ้น พร้อม hover effect
7. Price box แยกราคาให้เห็นชัด
8. Seller mini profile พร้อม Verified Seller visual badge
9. Badge สถานะพร้อมขาย/ขายแล้ว และ Escrow
10. Empty state และ skeleton loading
11. Responsive layout สำหรับมือถือและแท็บเล็ต
12. Dark-mode compatible styling

การตรวจสอบ:
