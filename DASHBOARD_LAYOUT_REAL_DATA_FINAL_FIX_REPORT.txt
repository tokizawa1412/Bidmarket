BidMarket Dashboard Layout Real Data Final Fix

แก้ไขใน public/index.html:
- ปรับ Header: ช่องค้นหาอยู่กึ่งกลางจริงและจำกัดความกว้าง ไม่ยาวเกินไป
- ปรับโลโก้มุมซ้ายบนให้แสดงเต็มภายในพื้นที่
- ปรับ Dashboard เป็น 3 คอลัมน์เดียวกัน: Sidebar ซ้าย / Content กลาง / Sidebar ขวา
- ปรับหัวข้อหลักกลางให้แสดงเฉพาะไอคอนค้อน/บ้าน/คน ไม่แสดงข้อความ
- เพิ่มระยะห่างไอคอนหัวข้อหลักให้กว้างขึ้นและจัดวางสวยขึ้น
- หน้าโปรไฟล์ใช้ Content กลางแบบเดียวกับหน้าอื่น ไม่ใช้ Layout แยกที่ทำให้ไม่เท่ากัน
- หน้าโปรไฟล์ลบ Sidebar เพื่อน/กิจกรรมภายในหน้าออก เพื่อไม่ให้ซ้ำกับ Sidebar ขวาหลัก
- หน้าบัญชีผู้ใช้แสดงหัวข้อย่อยเฉพาะ บัญชีผู้ใช้ / คอลเลคชั่น
- หัวข้อย่อยประมูลแสดงเฉพาะในกลุ่มประมูลเท่านั้น
- Sidebar ขวาเปลี่ยนจากข้อมูลตัวอย่างเป็นข้อมูลจริงจาก /api/profiles/:id
- ปรับการ์ด VIP ให้แสดงเต็ม ไม่ล้นขอบพื้นที่แสดงบัตร

ตรวจสอบ:
- node --check server.js ผ่าน
- node --check public_inline.js ผ่าน
